Minesweeper Game
@author Aldo Azali - 13516125

Deskripsi:
Permainan akan disetting secara default ke level Easy saat pertama kali dijalankan.
Untuk memulai permainan baru, maka dapat memilih opsi "New Game" pada bar menu "Game".
Untuk keluar dari permainan, maka dapat memilih opsi "Exit" pada bar menu "Game".
Untuk mengatur tingkat kesulitan permainan, maka dapat memilih pada bar menu "Options".
Tingkatan kesulitan permainan terdiri dari 4, yaitu Beginner(Easy), Intermediate(Medium), Expert(Hard), dan Custom.
Pada tingkat Easy, jumlah baris dan kolom yaitu 8 dan jumlah bom yaitu 10.
Pada tingkat Medium, jumlah baris dan kolom yaitu 16 dan jumlah bom yaitu 40.
Pada tingkat Hard, jumlah baris yaitu 32, jumlah kolom yaitu 16 dan jumlah bom yaitu 99.
Pada tingkat Custom, Anda dapat memilih jumlah baris, jumlah kolom dan jumlah bom sesuai minat Anda.

Catatan :
Pada tingkat Custom, Aplikasi diasumsikan selalu menerima masukan berupa angka Integer.
